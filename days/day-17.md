toscalar() scalar function returns a scalar constant value of the evaluated expression.

# Official Document

[toscalar()](https://learn.microsoft.com/en-us/kusto/query/toscalar-function?view=microsoft-fabric)

# Note

This scalar function is commonly used within complex queries, which require staged calculations. In the official document, this function was used in tandem with new_guid() to generated static guid in one example.
