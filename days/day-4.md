# Day 4

Project operator helps shape the output data schema. There are some variant operators as listed below:

- project (only show the specified columns or manipulated ones)
- project-away (only show the columns that are not specified)
- project-by-names (only show the specified columns)
- project-keep (only show the specified column names or patterns)
- project-rename (rename columns)
- project-reorder (rearrange the columns)
